3. A ( 1，1，0 )，B ( 2，1，1 )，C ( 1，1，1 )，D ( 1，3，2 )，E ( 2，0，1 ) 通過E的一直線L，分別交，於P，Q兩點，求P，Q的坐標。

答案：AB= ( 1，0，1 )，CD= ( 0，2，1 )
令P = ( p + 1，1，p )，Q ( 1，2 q + 1，q + 1 )
則EP = ( p – 1，1，p – 1 )，EQ = ( – 1，2 q + 1，q )
∵E，P，Q三點共線　∴q = – 1，EQ = ( – 1，– 1，– 1 )，p – 1 = 1，p = 2
P之坐標為 ( 3，1，2 )，Q的坐標為 ( 1，– 1，0 )
